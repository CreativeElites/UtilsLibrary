@JvmOverloads inline fun Context.showDialog(themeId: Int = 0, cancelable: Boolean = false, cancelableTouchOutside: Boolean = false, builderFunction: Builder.() -> Any): Unit

Shows AlertDialog

Parameters

themeId - Theme Id for Dialog

cancelable - Whether dialog is cancelable or not, by default dialog is not cancelable, use true to make it cancelable

cancelableTouchOutside - Whether dialog is cancelled on touched outside or not, by default it's not cancelable. Use true to make it cancelable when user touches outside of dialog

builderFunction - Pass your AlertDialog.Builder functions which helps you to create dialogs
